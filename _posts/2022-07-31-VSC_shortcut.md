한문장으로 HTML코드 만드는법 예시
```
nav>ul
nav안에 ul이 있다는 뜻

nav>ul>li*4

<nav>
    <ul>
        <li></li>
        <li></li>
        <li></li>
        <li></li>
    </ul>
</nav>
```

- 모든 상황에서 공통으로 사용하는 단축키  

|단축키|설명|
|fn + delete키|delete키(오른쪽 문자 삭제)|
|command + ,|앱 설정|
|command + TAB|다른 프로그램 선택|
|command + W|창(탭)닫기|
|command + Q|프로그램 종료|
|fn + ↑|페이지 위로|
|fn + ↓|페이지 아래로|
|command + ↑|최상위로|
|command + ↓|최하위로|
|command + ←|문단 맨 앞으로|
|command + →|문단 맨 뒤로|
